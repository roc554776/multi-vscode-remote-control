# 拡張機能の開発

## 拡張機能の（再）インストール

拡張機能を更新して VSCode に反映する際は、必ず `scripts/reload-extension.py` を使うこと。

### スクリプトの実行方法

```bash
cd /path/to/vcc-remote-control
python3 scripts/reload-extension.py
```

### スクリプトが行う処理

このスクリプトは、拡張機能の更新を確実に反映するために以下の一連の処理を自動化します：

1. **ビルド**: TypeScript コードをコンパイル
2. **パッケージング**: VSIX ファイルを生成
3. **VSCode 起動確認**: VSCode が起動していることを確認（未起動なら起動）
4. **アンインストール**: 既存の拡張機能を削除
5. **VSCode 再起動**: VSCode を完全に停止して再起動
6. **インストール**: 新しい VSIX をインストール
7. **検証**: `test-ping.py` で拡張機能が正常に動作しているか確認

手動でのインストールやリロードは、VSCode のキャッシュや状態管理の問題で正しく反映されないことがあります。  
**拡張機能を更新して VSCode に反映する際は、必ずこのスクリプトを使用してください。**
