# Milestone 1: タブ操作の自動化

## 背景・課題

VSCode Copilot Chat を自動操作するためのツールキットを構築したい。
その第一歩として、VSCode のタブを外部から操作できる仕組みが必要。

## やりたいこと

### 1. 外部プログラムから VSCode と通信したい
- VSCode 内部の情報を外部から取得したい
- 外部からコマンドを送信して VSCode を操作したい

### 2. タブの情報を取得したい
- 現在開いているタブの一覧を知りたい
- 各タブのファイルパスやタイトルを取得したい

### 3. タブを操作したい
- 指定したタブを閉じたい
- （将来的には：タブを開く、アクティブにする、など）

### 4. 動作確認を容易にしたい
- 実装が正しく動作しているか簡単に確認できる手段が欲しい
- 手順が複雑なら自動化したい

## 優先度

1. 通信基盤の確立（最重要）
2. タブ情報の取得
3. タブを閉じる操作
4. 動作確認の自動化
