# vcc remote control 価値提案 - RFP

## 背景

既存の remote control 手段で VSCode Copilot Chat の多くは操作できるが、実運用で求める体験には届いていない。
Copilot Chat を中心とした利用シーンで、より一貫して強力に扱える状態を目指したい。

## 問題

- VSCode Copilot Chat の操作は一部リモート化できても、必要な操作を十分に網羅できないことが問題。
- PC 前提の操作に寄っており、モバイルから同等の操作体験を得にくいことが問題。
- Copilot Chat が表示する情報や操作対象を、遠隔側でそのまま扱いにくいことが問題。
- 強力な操作を実現しようとすると安定性が下がりやすく、安定性と操作性の両立が難しいことが問題。

## やりたいこと

- VSCode Copilot Chat との連携に特化した remote control 体験を実現したい。
- モバイル環境からでも Copilot Chat を遠隔操作できるようにしたい。
- Copilot Chat が表示する要素を遠隔側で確認・操作できるようにしたい。
- 強力な操作性を持ちながら、継続利用できる安定した体験を両立したい。
